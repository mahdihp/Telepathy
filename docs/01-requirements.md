# Message Platform

## Goal

Build a production-grade real-time messaging platform to learn distributed systems, ScyllaDB, event-driven architecture, and high-performance backend development with Go.

The project is designed as a portfolio project rather than a commercial messaging application.

---

# Objectives

- Learn ScyllaDB
- Learn WebSocket at scale
- Learn Event Driven Architecture
- Learn gRPC
- Learn Distributed Systems
- Learn Redis
- Learn NATS JetStream
- Learn Docker & Kubernetes
